线性模型

假设一个实例有 ddd 个特征，即 x=(x1;x2;...xd)\mathbf{x} = (x_1;x_2;...x_d)x=(x1​;x2​;...xd​) ，则线性模型学习得到一个特征的线性组合，用以预测。

f(x)=w1x1+w2x2+...+wdxd+bf(\mathbf{x}) = w_1x_1+w_2x_2+...+w_dx_d+b f(x)=w1​x1​+w2​x2​+...+wd​xd​+b

注意:

(x1,x2,...,xn)(x_1,x_2,...,x_n)(x1​,x2​,...,xn​) 代表 nnn 维行向量，【以 ‘,’ 分隔】

(x1;x2;...;xn)(x_1;x_2;...;x_n)(x1​;x2​;...;xn​) 代表 nnn 维列向量，【以’;’ 分隔】

则，用向量形式表示为：

f(x)=wTx+bf(\mathbf{x}) =\mathbf{w}^\mathrm{T}\mathbf{x} +b f(x)=wTx+b

确定参数 w\mathbf{w}w 和 bbb 后，此模型就得以确定

线性回归使用线性模型预测实值输出标记

当样本特征只有一个时，称为一元线性回归 ，即 简单线性回归

我们先讨论一元线性回归这一简单情况：

寻找一条直线，最大程度地“拟合”样本特征和样本输出标记之间的关系

简单线性回归拟合方程

假设最佳拟合的方程为

y=ax+by=ax+b y=ax+b

则对于每个样本点 x(i)x^{(i)}x(i)，根据直线方程，得到预测值为

y^(i)=ax(i)+b\hat{y}^{(i)} = ax^{(i)} +b y^​(i)=ax(i)+b

我们把每个样本点 x(i)x^{(i)}x(i) 的真实值 记作 y(i)y^{(i)}y(i) ，则我们希望 y(i)y^{(i)}y(i) 和 y^(i)\hat{y}^{(i)}y^​(i) 的差距尽可能的小

通过式子表达两者差距：

  1. y(i)−y^(i)y^{(i)}-\hat{y}^{(i)}y(i)−y^​(i)
  2. ∣y(i)−y^(i)∣|y^{(i)}-\hat{y}^{(i)}|∣y(i)−y^​(i)∣
  3. (y(i)−y^(i))2(y^{(i)}-\hat{y}^{(i)})^2(y(i)−y^​(i))2

这三种结果都可以表达两者差距，那么哪一种合适呢？ ====> ③

​ 若使用第一个，可以很容易看出其不合理性，如果一个预测值比真值多了5，另一个预测值比真值少了5，两者得到的 +5+5+5 和 −5-5−5 相互抵消为 000 ，这显然是不合理的。

​ 因此我们想到，把第一个式子加上绝对值，这样是不是就合理了呢？貌似是合理了，但是绝对值函数并不是处处光滑的，即有不可导点，因此用第二个式子作为差距的表达时，则会丧失一些数学特性。

​ 所以使用第三个式子描述预测值与真值的差距很合适

简单线性回归目标

使 ∑i=1m(y(i)−y^(i))2\sum_{i=1}^{m}(y^{(i)}-\hat{y}^{(i)})^2∑i=1m​(y(i)−y^​(i))2 尽可能小

即寻找 a、ba、ba、b 的值，使得 ∑i=1m(y(i)−ax(i)−b)2\sum_{i=1}^{m}(y^{(i)}-ax^{(i)}-b)^2∑i=1m​(y(i)−ax(i)−b)2 尽可能小

(a∗,b∗)=arg⁡min⁡(a,b)∑i=1m(y(i)−ax(i)−b)2(a^*,b^*) = \underset{(a,b)}{\arg\min}\sum_{i=1}^m (y^{(i)}-ax^{(i)}-b)^2 (a∗,b∗)=(a,b)argmin​i=1∑m​(y(i)−ax(i)−b)2

∑i=1m(y(i)−ax(i)−b)2\sum_{i=1}^{m}(y^{(i)}-ax^{(i)}-b)^2∑i=1m​(y(i)−ax(i)−b)2 被称为 损失函数 (Loss Function)

参数学习算法基本思路

  • 通过分析问题，确定问题的损失函数
  • 通过最优化损失函数，获得机器学习的模型

问题解决

最小二乘法

设损失函数为 J(a,b)J(a,b)J(a,b):

J(a,b)=∑i=1n(y(i)−ax(i)−b)2J(a,b) = \sum_{i=1}^{n}(y^{(i)}-ax^{(i)}-b)^2 J(a,b)=i=1∑n​(y(i)−ax(i)−b)2

则令 ∂J(a,b)∂a=0、∂J(a,b)∂b=0\frac{\partial J(a,b)}{\partial a} = 0 、\frac{\partial J(a,b)}{\partial b} = 0∂a∂J(a,b)​=0、∂b∂J(a,b)​=0

因为 bbb 简单,先算 bbb:

∂J(a,b)∂b=∑i=1m2(y(i)−ax(i)−b)(−1)=0\frac{\partial J(a,b)}{\partial b} = \sum_{i=1}^{m} 2(y^{(i)}-ax^{(i)}-b)(-1) = 0 ∂b∂J(a,b)​=i=1∑m​2(y(i)−ax(i)−b)(−1)=0

约掉系数:

∑i=1m(y(i)−ax(i)−b)=0\sum_{i=1}^{m} (y^{(i)}-ax^{(i)}-b) = 0 i=1∑m​(y(i)−ax(i)−b)=0

b=yˉ−axˉ{\color{Red} b=\bar{y}-a\bar{x}} b=yˉ​−axˉ

∂J(a,b)∂a=∑i=1m2(y(i)−ax(i)−b)(−x(i))=0\frac{\partial J(a,b)}{\partial a} = \sum_{i=1}^{m} 2(y^{(i)}-ax^{(i)}-b)(-x^{(i)}) = 0 ∂a∂J(a,b)​=i=1∑m​2(y(i)−ax(i)−b)(−x(i))=0
